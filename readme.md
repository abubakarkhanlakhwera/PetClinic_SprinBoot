# Spring PetClinic Practice

Spring PetClinic practice project for Jenkins and Spring Boot learning.

## Purpose

Spring PetClinic is a familiar sample app, making it useful for learning Spring Boot structure and CI/CD without needing a new domain. This repo is mainly for framework and pipeline practice.
